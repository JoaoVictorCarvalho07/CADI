# Documentação de User Stories

## 6. User Stories

### 6.1 Aluno / Responsável

US-1.1: Como aluno ou responsável, quero consultar a frequência escolar, para acompanhar a participação do aluno nas atividades.

US-1.2: Como responsável, quero atualizar meus dados de contato, para garantir comunicação eficiente com a instituição.

US-1.3: Como aluno ou responsável, quero visualizar comunicados institucionais, para me manter informado sobre mudanças importantes.

---

### 6.2 Professor

US-2.1: Como professor, quero registrar a frequência dos alunos, para manter o controle pedagógico da turma.

US-2.2: Como professor, quero elaborar pareceres descritivos individuais, para acompanhar o desenvolvimento dos alunos.

US-2.3: Como professor, quero enviar notificações relacionadas a minha oficina para os alunos e pais matriculados nela, para informar imprevistou ou especificações necessarias para a proxima oficina como materiais.

---

### 6.3 Secretario(a)

US-3.1: Como secretário acadêmico, quero cadastrar novos alunos, para manter os registros atualizados.

US-3.2: Como secretário acadêmico, quero vincular/remanejar alunos às turmas, para organizar a estrutura acadêmica.

US-3.3: Como secretário acadêmico, quero emitir declarações e certificados, para atender solicitações formais.

US-3.4: Como secretário academico, quero visualizar(talvez modificar) pareceres descritivos individuais, para acompanhar o desenvolvimento dos alunos.

US-3.5: Como secretário administrativo, quero visualizar relatórios institucionais, para apoiar decisões estratégicas.

US-3.6:Como secretário, quero cadastrar e gerenciar novas turmas, para atender a demanda de alunos e horarios disponiveis.

US-3.7: Como Secretario, quero poder anexar documentos e relacionalos com algum usuario, para manter registro oficial do usuario

US-3.8: Como Secretario, quero poder enviar notificações para contextos diferentes (turma especifica, aluno especifico, professor especifico, toda a instituição)
---

### 6.4 Gestão Administrativa

US-4.1: Como gestor administrativo, quero visualizar relatórios institucionais, para apoiar decisões estratégicas.

US-4.2: Como gestor administrativo, quero acompanhar dados financeiros de forma consolidada, para avaliar a saúde financeira da instituição.

---

### 6.5 Financeiro

US-5.1: Como responsável financeiro, quero registrar contas a pagar e a receber, para controlar o fluxo de caixa.

US-5.2: Como financeiro, quero emitir notas fiscais, para cumprir obrigações legais.

---

### 6.6 Psicólogo

US-6.1: Como psicólogo, quero acessar apenas dados essenciais do paciente, para preservar o sigilo profissional.

US-6.2: Como psicólogo, quero registrar observações terapêuticas confidenciais, para acompanhar a evolução dos atendimentos.

---
